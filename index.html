GitHub Pages免费的静态站点，三个特点：免费托管、自带主题、支持自制页面和Jekyll。

为什么使用Github Pages

  1. 搭建简单而且免费；
  2. 支持静态脚本；
  3. 可以绑定你的域名；
  4. DIY自由发挥，动手实践一些有意思的东西git,markdown,bootstrap,jekyll；
  5. 理想写博环境，git+github+markdown+jekyll；

两种pages模式

  • User/Organization Pages 个人或公司站点
  • Project Pages 项目站点
